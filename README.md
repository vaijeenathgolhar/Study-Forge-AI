# Study Forge AI

Study Forge AI is a multi-agent AI learning assistant that helps students with coding, research, note generation, and resume analysis.

## Features

* AI Chatbot
* Coding Assistant
* Learning Assistant
* Notes Generator
* Research Tool
* ATS Resume Checker

## Tech Stack

* Python
* Streamlit
* LangChain
* Groq LLM

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py
